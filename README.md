## 参与开发

### 克隆项目到本地

除非您很确定本版本的 PCL 存在问题，否则我们推荐您使用上游分支的版本进行开发。

```bash
git clone git@github.com:Hex-Dragon/PCL2.git
```

### 使用 Visual Studio 开发

你可以在[此处下载 Visual Studio](https://visualstudio.microsoft.com/)。

使用 Visual Studio 打开克隆下来的仓库中的 `Plain Craft Launcher 2.sln` 文件。
